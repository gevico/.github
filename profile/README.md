# 格维开源社区(GTOC)

格维开源社区（Gevico Tech Open Community，简称 GTOC ），社区名称寓意“格物致知、多维创新、技术包容、社区开放”，以分享嵌入式、操作系统、编译器、模拟器、虚拟化技术等方面开源项目为目标，传播开源技术，布道开源文化。
